## How to use

- Create a `.env` file and copy the following and fill values as required:
  ```env
  JIO_ID=""     # Your JioNet ID
  PASSWORD=""   # Your stupid ahh password

  # The JioNet service at our college has brainrot preinstalled so just keep the value below as it is  
  ACCOUNT="******"
  ```
- Install required packages and run the application. Do as you'd normally do or follow below:
  ```bash
  uv venv
  source .venv/bin/activate  # On Windows: .venv\Scripts\activate
  uv pip install -r requirements.txt
  python anya.py
  ```
